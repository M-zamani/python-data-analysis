## Project Overview
This project analyzes customer churn using a telecom-style dataset to identify key drivers of churn and provide actionable business recommendations.

**Goal:** Understand why customers leave and suggest retention strategies.  
**Tools:** Python (Pandas, NumPy, Matplotlib, Seaborn), Jupyter Notebook, Power BI / Tableau

---

## Dataset

Recommended dataset (Kaggle):
**Telco Customer Churn**
https://www.kaggle.com/blastchar/telco-customer-churn

Download `WA_Fn-UseC_-Telco-Customer-Churn.csv` and place it in the project folder as:

```
data/churn.csv
```

---

## Project Steps

1. Data loading and inspection
2. Data cleaning (missing values, types, encoding)
3. Exploratory Data Analysis (EDA)
4. Churn driver analysis
5. Business insights and recommendations

---

## Key Questions Answered

- Which customer segments churn the most?
- Does contract type affect churn?
- How does tenure relate to churn?
- Which services increase churn risk?
- Which payment methods are riskier?

---

## Outputs

- Jupyter Notebook with full analysis
- Visualizations for churn trends
- Clean dataset ready for Power BI / Tableau dashboard

---

## Dashboard Ideas (Power BI / Tableau)

KPIs:
- Churn Rate (%)
- Avg Tenure (Churned vs Active)
- Monthly Charges by Churn

Charts:
- Churn by Contract Type
- Churn by Internet Service
- Tenure Distribution by Churn
- Payment Method vs Churn

---

## Business Recommendations (Example)

- Offer discounts to month-to-month customers after 3 months
- Promote annual contracts to high-risk segments
- Improve service quality for fiber internet customers
- Incentivize auto-payment methods
